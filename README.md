# Retail-Sales-Profit-Insights-Power-BI-Dashboard
Interactive Power BI dashboard analyzing retail sales, profit, and customer segments. Includes KPI tracking, region and category-wise profit analysis, shipping cost insights, and dynamic filters. Built using Power BI, DAX, and data modeling to transform raw data into actionable business insights.
